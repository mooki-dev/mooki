# Mooki

## Prérequis

- Node.js (version 18 ou supérieure)
- npm (version 8 ou supérieure)

## Installation

1. Clone le dépôt :
   ```bash
   git clone <url-du-depot>
    cd mooki
    ```
2. Installe les dépendances :
   ```bash
   npm install
   ```
3. Lance le serveur de développement :
   ```bash
   npm run docs:dev
   ```
